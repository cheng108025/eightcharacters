# 由八字看流年走運(流年走十神)

> 出處:《學八字,這本最好用》第六章(掃描頁 69–83)

## 總論:流年怎麼看十神

- 看流年運勢,是把「流年天干、地支」對照日主換算成十神,再看該十神當年的動態現象。
- **天干管上半年,地支管下半年**:例如某年天干換算為七殺、地支換算為食神,則上半年走七殺運、下半年走食神運(書中命盤 22 即此例)。
- 流年十神的論斷還要配合:身強身弱、該十神是喜用神或忌神、本命盤中原有哪些十神(產生特殊組合)、有無被流年合住等條件,吉凶才會準確。
- 以下各節論斷可作為批八字流年之參考,實際論命請搭配本命盤整體結構。

## 比肩(流年走比肩運)

### 基本現象
- 錢財容易被「劫」:今年錢盡量不要借人、不要跟會、不要合夥,錢財易被劫走,也常會掉東西。
- 外表看起來樂觀,內心卻有些想不開,偶爾會冒出輕生念頭;做事較憑直覺、易有輕率的念頭。
- 感情上容易有困擾,剪不斷理還亂;人情包袱重、耳根軟;不喜歡的人對自己再親近也不予理會,生氣多悶在心裡,遇到仇人乾脆不理會。
- 個性不服輸,所以沒朋友聊天打屁會很難受、鬱悶;朋友多、聚在一起就很高興快樂。
- 今年本身擁有的東西會與人分享共有,也容易出現與朋友合作事業的機會。
- 今年的心態是「感覺爽就好」,有沒有賺錢沒關係;不會害人,但有時會被害。老朋友會出現或再見面。
- 今年會顯得特別重視朋友、兄弟姊妹——只顧朋友,老婆會怨嘆喔!

### 走比劫年(比肩+劫財)的共同注意
- 錢財要控制得當,否則容易流失;事情不來則已,一來就接二連三,有招架不住之感,此時「儲糧」(存錢備糧)最重要。
- 失戀機會特別大,而且是刻骨銘心的失戀;親愛的人一不小心就會變成別人的。

### 依喜忌與得分的差異
| 條件 | 現象 |
|---|---|
| 比肩為喜用,且在命局中 9~15 分 | 擇善固執,不輕易改變立場;做事踏實,較能逐步達成目標 |
| 比肩為喜用神,且比肩有 10~13 分 | 會受到兄弟、朋友、同行的幫助 |
| 比肩為忌神,且命局中得分超過 17 分以上 | 比較會堅持己見,與人相處常有意見衝突,不太會體諒他人 |
| 比肩落空亡 | 兄弟緣薄、兄弟少,也不易受兄弟之助力 |

### 身強身弱差異
- **比肩多且身強者**:若要換職業,應以自由業或能自己作主的工作為目標。
- **身強格又走比肩、有運但無財星**:相當自我、以我為主,但以自我為中心的判斷未必正確;今年會很難過,錢財不夠花。
- **身強行比肩**:開運即能改變,可借用他人之運而得財;今年會很難過、錢財不夠花的情況可因開運改善。
- **身強行比肩的凶象**:易有官訟、破財、刑妻、不順之事,也易遭朋友、合夥人拖累,有「公親變事主」之嘆。
- 今年需注意父親身體;本身會比較勞碌、易徒勞無功,應存委曲求全、卑微低下之心,方有改變命運的可能。

## 劫財(流年走劫財)

### 基本現象(與比肩年類似但更烈)
- 錢不要借人、不要跟會、不要合夥,錢財易被劫走,常掉東西。
- 外表樂觀、內心想不開,有輕生念頭;較憑直覺做事、易有輕率念頭;感情困擾剪不斷理還亂,人情包袱重、耳根軟。
- 一個人獨處時感情的脆弱會全部湧現,是「人群中的孤獨者」。
- 個性不服輸,沒朋友打屁會很難受鬱悶;朋友多在一起就很高興快樂。
- 擁有的東西會與人分享共有,也易有與朋友合作事業的機會。
- 感覺爽就好,有無賺錢沒關係;不會害人、有時會被害;老朋友會出現或再見面。
- 容易有「分離」的感覺:如失戀、離婚等;較無安全感,怕流失機會、又怕會賠錢。

### 身強身弱差異
| 條件 | 現象 |
|---|---|
| 身強 | 花錢如流水,財來財去,常常追著錢跑、總是不夠花 |
| 身強(且財星弱時) | 思維清晰、能言善道、善於察顏觀色,應變力好、富社交能力,懂得借力使力 |
| 身強(財星弱) | 朋友間感情不錯,又能得兄弟姊妹、同輩間的幫助 |
| 身弱 | 可能辛苦賺三年所得,不用幾個月就虧光光 |
| 身弱 | 朋友間易有是非,或受朋友拖累;財務往來必須注意;戀人易被爭奪——愛的人自己反而羞於說出口,結果被別人追走了 |
| 身弱(有財星) | 今年不宜經商、擴大投資規模或改做投機行業,千萬不可合夥,錢拿出去就拿不回來了 |
| 身強(有財星) | 今年會有財源廣進的現象,凡事順利之象;但身弱者逢財反易破財,並須注意身體健康 |

## 食神(流年走食神運)

### 基本現象
- 今年特別靜得下來,適合研究及發明工作;學習能力特別強,吃的機會也特別多。
- 吃的機會多,但要注意消化系統不太好。
- 才華如平時施展不開,今年做事重邏輯、表現慾強、理解力強;但多學不專、急性子、喜自由、易被感動;文武兼備,卻很難專精。

### 子息與女命
- **女命走食神運**:女命以食神、傷官論子息;想生孩子的話,走食神年懷孕機會較多,今年好好加油。
- **流年走食神命帶傷官者(女命)**:女命易流產、不易生男生,也易患婦女病及腫瘤。

### 特殊組合
| 組合 | 現象 |
|---|---|
| 食神被合 | 才華不能展現、決策易失誤、身體狀況差;嚴重時生命會有危險,亦會危及子女 |
| 流年走食神、逢偏印(梟印奪食) | 才華施展不開,做事常前功盡棄、常遭陷害、易被扯後腿(自己也喜歡扯人後腿);一生多敗少成,常逢意外之災、血光;內心較鬱悶,會誤人誤己 |
| 流年走食神本命有偏印 | 今年小人出動,做事常前功盡棄、生命亦憂;做事無思考空間、太直接、欠缺考慮 |
| 流年走食神本命有七殺 | 今年的磁場「想凶也凶不起來」,權勢好像被制住了(食神制殺) |
| 流年走食神命帶偏印(男命) | 易有隱疾復發而發生致命的危險 |

### 身強身弱差異
- **身強**:今年特別靈光,跟朋友相處特別友善、特別會講話,反應靈敏有巧思。消化系統顯得特別好、口慾佳、挑食,注意體型會發胖;最喜嘗鮮,男命易喜女色。今年易成名,有很多事會被表揚或讚賞。
- **身弱**:注意身體毛病;缺乏活動力、性情不佳、沒有堅持力,易逞口舌之快。
- **女命身弱**:要注意身體狀況,疾病今年特別容易上身;女命易懷孕。

## 傷官(流年走傷官)

### 基本現象
- 今年很有才華,但博學不精;頭腦好、學習能力強、追求完美,但缺乏耐性。
- 事事爭第一,沒爭到第一就很難過;但「一爭第一就出事」,須學做老二哲學。愛受人誇讚、不喜被批評,貪做易虧錢。
- 今年做事會覺得特別不順,要學「潛水艇哲學」,盡量不要強出頭;考運差。
- 男孩子帶傷(本命有傷官)又走傷官:情緒、脾氣會特別差、特別壞,往事常環繞心中。
- 今年可能會傷財、傷身,投資能免則免。

### 婚姻感情
- **女命走傷官**:因愛丈夫而又想管丈夫,做事貪做、明念暗念、句句傷丈夫的心,俗稱「妻管嚴」。俗話說:一丈之內是丈夫,一丈之外馬馬虎虎——想婚姻好,嘴巴請儘量少說刺激的話。
- **女命走傷官**:未婚時會想結婚,婚後又想離婚,容易有婚變(傷官剋正官=夫星)。
- **男命走傷官**:今年會特別龜毛、白目,奇招特別多;男命日主座下帶傷官,跟妻子不好溝通。
- 女命注意與先生或男朋友間感情易有波動。
- **走傷官**:男命可情場得意;女命易懷胎生子。
- **身強走傷官(男命)**:注意跟兒女不合;若無生兒育女打算,應特別注意避孕措施。

### 特殊組合
| 組合 | 現象 |
|---|---|
| 流年走傷官、命中又有食神 | 從事教學或組織工作,可得桃李滿天下 |
| 傷官通根八字(天干也有傷官) | 今年與兄弟無緣,做再多也無人欣賞。傷官通根在哪一柱,該時期會受到重大創傷或過錯:年柱=幼年、月柱=青年期、日柱=中年期、時柱=老年期 |
| 傷官見本命有正官(傷官見官) | 今年會有血光、官司;千萬不要作保,做事會出爾反爾,較難成功 |
| 傷官被流年合 | 處理事情或子女成長問題時思緒較雜亂,才華點子不現、處事不明、有點迷迷糊糊;需多問幾個人再做決定 |

### 身強身弱差異
- **身強**:對社會較有不滿情緒,是標準的批評家;但做事有魄力,一定完成到底。顯得特別聰明,很在意別人的肯定與讚美;富感性、喜抒發情緒。六親方面顯得更無話可談、易有紛爭。
- **身弱**:受你幫助最多的人,往往也是最容易陷害你的人;才華特多但流於雕蟲小技,難成大事之嘆——明年就會好轉。來自家庭、學業或工作的壓力感覺特別大。想轉換跑道者,可選擇自由業或個人工作室之類。
- **身弱**:思想比較偏激、行事任性、說話誇大。生理需求較強、性觀念較開放,要注意因情慾惹禍;注意血光、官司、破財、車關。

## 正財(流年走正財)

### 基本現象
- **走正財年**:行動派,喜歡熱鬧的地方,比較有賺錢機會,生意上人緣較佳。
- **走正財年**:工作上喜歡固定穩定(固定財源),做人比較節儉。
- **走正財年**:只想專心工作、也樂於工作,所以賺錢上感覺不那麼辛苦。
- **走正財年(未婚)**:今年會有結婚的機會;結婚最好「太太能助先生」,男生也較疼老婆。
- **財多又走財年**:男命很有異性緣、有歡喜心;女命則像是「錢嫂」(整天為錢忙)。

### 特殊組合
| 組合 | 現象 |
|---|---|
| 流年正財合日主或月支 | 特別注意錢財與身體方面會有損失;女命夫妻感情會不好 |
| 流年走正財、本命有正印+比肩、劫財 | 注意破財、損身;有購置不動產的可能。男命:太太與母親較不合,會有婆媳問題;未婚者可娶妻 |
| 正財被流年合 | 今年錢財流失大(男命太太若亦被合,嚴防婚變);又遇本命有「偏印+食神」和「傷官+正官」者,會投資失敗 |
| 流年走正財、本命有比肩及劫財 | 注意生命與身體健康;財多身弱,「財多逢劫必破」,有了錢反而吃藥多 |
| 太歲合財(男命) | 未婚者容易失戀;已婚者錢財流失大,小心老婆身體及婚變、了錢賠錢 |

### 身強身弱差異
- **身強**:只要意志力夠又有耐性,想賺錢並不難。
- **身強(本命無正財、偏財)**:宜從事勞力密集性質行業,有過分保守之嘆。
- **身弱**:為人不要太小心眼,要懂得變通才會賺到錢;對錢極為保守。
- **身弱**:今年特別注意會有破財或桃色糾紛,須開運。
- **身弱走正財**:本身條件尚可,今年可望做生意,工作上會有斬獲。
- **身弱逢財**:易破財,且須注意身體健康(身弱不任財)。

## 偏財(流年走偏財)

### 基本現象
- **走偏財**:比較不重財(看錢不重),賺錢快、虧錢也快;小錢不喜歡賺,不喜歡穩定、固定的工作,會挑剔,感情不專、有困擾。
- **流年走偏財、八字無正財偏財**:為人沒有金錢及數字觀念,財來財去,不知如何賺錢、不重錢財、沒有歡喜心。
- 跟朋友出門請客可以觀察:身強者付錢後不後悔,身弱者付錢後會後悔。
- **流年走偏財**:今年要特別注意父親身體狀況,可能會比以往差。

### 男女與桃花
- **女命帶七殺走流年偏財**:易養小男人(賺錢養男人)。
- **女命身弱走偏財**:視賺錢為壓力,賺得很辛苦;要注意仙人跳;計畫好像永遠趕不上變化。
- **男命身強、走偏財本命有七殺+偏財**:會有「名聲、地位」,但好色居多、養妾機率高,總是想著外面的女人。
- **流年走偏財、本命天干帶偏財**:外面帶妾不怕人知;地支帶偏財則做事會偷偷摸摸。

### 特殊組合
| 組合 | 現象 |
|---|---|
| 偏財被流年合 | 今年開支特別大,生意也會賠錢、錢財流失大;生意人有可能一敗塗地 |
| 太歲合偏財 | 特別注意:錢財損失、父親身體欠安、情人失戀、野桃花易被捉 |
| 偏財被合那一年 | 年紀大者表父親身體不好或病危;年紀輕者表遇仙人跳,或有妾被老婆逮到 |

### 身強身弱差異
- **身強**:特別慷慨豪爽,深諳人情世故,交際特別好。
- **身弱**:請你酒不要喝那麼多、要戒色、不要誇大其詞,錢花少一點。

### 逢財反易破大筆財(財年總結)
- **身強**:得正職;辛苦耕耘所累積的財富可保,亦主財運亨通;易有偏財運,「女人運」當道;男命今年會遇幼齒相戀機會。
- **身弱**:花費大;事業機會多但無福消受;注意「專精一藝」可發財。

## 正官(流年走正官)

### 基本現象
- 正官之人很好面子、喜歡別人讚賞;要責難他時不能太直接,而且旁邊不要有其他人。
- **身強**:俗話說「見官得官」,不得官也會得貴;多做好事,會有好報。
- **女命流年走正官**:今年比較重視丈夫,對先生的一舉一動也比較關心;社會地位有機會提升。

### 特殊組合
| 組合 | 現象 |
|---|---|
| 身強流年走正官、本命又有正官+七殺 | 在社會上「會有兩個頭銜」;官殺混雜,個性陰沈、神秘,一生有名望、地位 |
| 身弱流年走正官、本命沒有正官+七殺 | 今年是非特別多、壓力特別大,精神也易緊張 |
| 流年走正官、本命有傷官 | 恐有血光之災,注意名聲、地位會不保 |
| 流年走正官、命帶食神 | 行事顯得慵懶(不積極、沒動靜),做什麼事都覺得不帶勁 |
| 女命流年合正官 | 注意老公身體可能因某種原因變差;日主合正官者很重視老公 |
| 雙正官(命有兩個正官) | 較容易再婚;流年走到正官會想結婚 |
| 正官被流年合 | 職業上會有變動或被奪;女命應小心丈夫「可能會走掉」;今年不要出風頭、不要當老大 |
| 女孩子正官被流年合 | 注意先生的一舉一動,先生可能會有外遇 |
| 男孩子官被流年合 | 做任何事要特別注意,可能有名聲、地位掃地的情況發生 |
| 今年正官被合 | 要注意明年傷官年,表示會有官司纏身的情形 |
| 女命中有正官、七殺、又逢正官 | 歲運易有桃色糾紛或「爬出圍牆」(紅杏出牆)之現象,該注意 |

### 身強身弱差異
- **身弱**:來自家庭、學業或工作上的壓力感覺特別大;處事較優柔寡斷、欠考慮、猶豫。
- **身弱**:好像缺乏自信、魄力不足;最好找夥伴一同解決問題;常為了圓滿面子卻違背心意說謊。
- **身弱**:注意會有是非或降職丟官的事發生。
- **身弱者流年走正官**:今年身體不好且會變成體弱多病——因為「身弱不得任財官也」。

## 七殺(流年走七殺)

### 基本現象
- 今年有可能會有災難,如:意外、官司、血光方面,應多小心。

## 正印(流年走正印)

### 基本現象
- 今年所有的事好溝通,貴人會比較多,但人容易顯得慵懶。
- 比較喜歡用頭腦、喜靜、很會思考;不喜歡動、不是行動派;個性固執但心地軟。
- 主觀意識很強,但比較有慈悲心、有佛緣,會從事宗教活動。
- 分配事情會一視同仁、以大家意見為意見;個性固執,但遇到財(賺錢)則不固執——「財會壞印」。
- **流年走印時**:今年會突然很想購置不動產,也同時會有得到祖產的機會。

## 偏印(流年走偏印)

### 基本現象
- 今年心性不穩定,做事常三心二意,所以比較不易成功。
- 特別具有第六感,更善於察言觀色,喜怒不形於色。
- 很喜歡外出結緣,投資、花費較兇;對環境敏感度高但較無信心。
- 在想法上比較消極;很想買不動產,但能力上有所不足。
- 對事情的敏銳度好像都是後知後覺,但對已認知的事會非常主觀。
