# 天干五合(合化)

> 出處:《學八字,這本最好用》第二章第四節「以八字天干合化所產生的結果」(書頁 56–61,掃描圖 15–17)

## 什麼是天干五合

- 天干五合又稱「**鴛鴦之合**」:十個天干中,凡是相隔五位、彼此「剋而有情」的兩干即構成一組合(甲數到己為第五位,故稱五合)。
- 合化的基本結構:一方是另一方的「官」,另一方則是對方的「財」——官財互見、有情相合。
- 命中帶天干合的人,共通傾向:
  - 異性緣好、容易與人相處、能接納他人;
  - 但也往往**缺乏主見、容易隨波逐流**。

## 五組合化總表

| 組合 | 合化五行 | 別稱 | 官財關係 |
|------|--------|------|---------|
| 甲＋己 | 化土 | 中正之合 | 甲為己之官,己為甲之財 |
| 乙＋庚 | 化金 | 仁義之合(正義之合) | 庚為乙之官,乙為庚之財 |
| 丙＋辛 | 化水 | 威制之合 | 丙為辛之官,辛為丙之財 |
| 丁＋壬 | 化木 | 淫暱之合 | 壬為丁之官,丁為壬之財 |
| 戊＋癸 | 化火 | 無情之合 | 戊為癸之官,癸為戊之財 |

## 各組合化的性情特徵

### 甲己合化土——中正之合

1. 講信用、重承諾,但個性較固執。
2. 典型的好好先生:不會亂來,也不太會講好聽的奉承話。
3. 一板一眼,較不懂生活情趣、不解風情。
4. 人緣很好。

### 乙庚合化金——仁義之合

1. 重義氣、有仁德;個性剛強,果斷力強。
2. 剛強果斷之外,做事積極進取。
3. 不懂生活情趣,重視規律。
4. 對人和善,不會刻意得罪別人;就算看到不高興的事也會忍下來。

### 丙辛合化水——威制之合

整體印象:人緣好(也因此易有偷情的機會)、重情趣、氣質好、愛名牌、希望別人聽自己的。

1. 個性強,儀表帶威嚴。
2. 重情趣,重視心靈層面的感受。
3. 氣質好、很聰明。
4. 偏好年輕的異性。
5. 若地支逢沖,容易變成縱情縱慾。

### 丁壬合化木——淫暱之合

整體印象:心軟、注重穿著、較無心機、人緣好。

1. 重品味,風情萬種、婀娜多姿,心地善良。
2. 講究穿著打扮,異性緣非常好。
3. 重情調,懂得生活情趣。
4. 心慈性善,主長命多壽。

### 戊癸合化火——無情之合

整體印象:為人重情調,但翻臉像翻書;做人阿莎力(也因此比較可能有一夜情)。婚配上容易出現「女大男小」或年齡差距較大的組合。

1. 容易翻臉、易生氣,但氣過就算了。
2. 喜歡成熟型的異性,本身外型通常也不錯。
3. 重情趣但顯得無情,給人有點冷漠的感覺。
4. 切忌激怒他——一旦被激怒,說什麼都沒用。
5. 冷靜理智、禮節周到,但也比較現實。

## 命盤實例(命盤 4)

書中以一張 1980 年出生的女命為例:天干出現「合」的標示,並且天干中同時出現**三個庚**——由此帶出下一個主題「天干多字相同」的看法。

## 天干多字相同的現象

四柱天干若出現同一字連續多見(三同、四同),命上各有對應的現象:

### 天干字連三同

| 天干 | 三字相同的現象 |
|------|--------------|
| 甲 | 較易獨守空房 |
| 乙 | 災禍較難抵擋 |
| 丙 | 人老以後孤獨 |
| 丁 | 婚姻較難成局 |
| 戊 | 意外事件特多 |
| 己 | 多才多藝 |
| 庚 | 多財多祿 |
| 辛 | 較難昌榮 |
| 壬 | 家業興盛 |
| 癸 | 他鄉意外 |

### 天干字連四同

| 天干 | 四字相同的現象 |
|------|--------------|
| 甲 | 少妻少子 |
| 乙 | 多災多難 |
| 丙 | 子媳空缺 |
| 丁 | 壽命長久 |
| 戊 | 人緣孤立 |
| 己 | 為人忠厚 |
| 庚 | 遠走他鄉 |
| 辛 | 人可長壽 |
| 壬 | 人可富貴 |
| 癸 | 易遭火災 |

例:命盤 4 天干見三個庚 → 對照上表,主一生中「多財多祿」。

## 學習重點整理

1. 天干五合的口訣:**甲己土、乙庚金、丙辛水、丁壬木、戊癸火**。
2. 每組合都可從「官/財互換」理解:合中一方管束另一方(官),另一方是對方所求(財),所以合是「剋而有情」。
3. 有合者人際圓融、異性緣好,但代價是主見不足。
4. 判斷性格時先看有無天干合,再查天干是否有三連同、四連同的特殊現象。
