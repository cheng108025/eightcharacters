# 四馬、四花、四庫

> 來源:《學八字,這本最好用》第四章第三節(掃描圖 44–47,書頁 114–121)。
> 主旨:十二地支可分成三類——馬(驛馬)、花(桃花)、庫(財庫)。從命盤四個地支中數「幾馬、幾花、幾庫」,即可看出一個人一生的驛馬貴人運、桃花人緣運與財庫運。

## 二、四馬:寅、申、巳、亥(驛馬星)(圖 44)

驛馬主動能、衝勁、奔波、外務。依命盤地支中馬的數量論斷:

| 馬數 | 論斷 |
|---|---|
| 無馬 | 沒有衝勁,不適合當業務;適合內勤工作(或家庭主婦),衝勁弱 |
| 一馬 | 好動、適合做業務;願意跑、愛跑,但跑不遠;靈機應變、不受拘束 |
| 二馬 | 喜橫衝直撞;愛旅遊、搬遷、變動;有行動力、執行力;好動閒不住、不受拘束;業務高手、不常留在屋內;注意車關。若沒有合來收,只會亂衝、不懂收成,為錢忙來忙去 |
| 三馬 | 藝高膽大、勞碌奔波;什麼都要闖、多車關、居無定所四處跑 |
| 四馬 | 家裡待不住、常往外跑,一出門就不知回家;會設定目標後馬不停蹄;藝高人膽大;開運上宜設定明確目標收束 |

驛馬互沖的補充:

- **寅申沖**:閒不住;開車快、愛走大馬路;有車關、易生車禍,手腳較會有問題。
- **巳亥沖**:辯才無礙、口才好、很會辯;追根究底;開車較會鑽小巷;多情、愛管閒事;注意車關(書中建議以「開運小羅盤」避車煞——屬作者的開運商品說法,參考即可)。

## 三、四花:子、午、卯、酉(桃花——異性緣與人際關係)(圖 45–46)

桃花主人緣、異性緣、外貌魅力。依花的數量論斷:

| 花數 | 論斷 |
|---|---|
| 無花 | 很嚴肅、臉臭;異性緣差、人際關係不良 |
| 一花 | 有人緣、早熟、貌美,異性緣還可以 |
| 二花 | 真有人緣、早熟、漂亮貌美,異性緣不錯 |
| 三花 | 異性緣重、早熟、適合公關;人緣佳、異性緣特別好;文才風流、善於應對(也好爭辯) |
| 四花 | 兩極:超有人緣(但要注意道德觀、慎防濫桃花、處處留情,為求圓融不惜說謊),或超沒人緣(過度自戀、孤芳自賞);早熟、不太愛理人,帶雙重人格傾向 |

桃花互沖的補充:

- **子午沖**:情緒不穩定、脾氣不好、個性極端;人緣很好、很有異性緣;容易腦神經衰弱。
- **卯酉沖**:很敏銳、頗具第六感、眼睛銳利;人緣好、異性緣佳;較龜毛、有潔癖、做事有計畫;背脊常感覺陰涼,易受「陰」干擾——宜避開陰喪事物、不進陰廟、少碰古物及陰氣重之物。

### 花所在柱位的論斷(圖 46)

書中以「時日月年」四格、□ 與「花」標示位置(右為年、左為時):

| 花的位置 | 論斷 |
|---|---|
| 花在年支 | 有長上緣,或母親有人緣 |
| 花在月支 | 自己漂亮或有人緣 |
| 花在日支 | 配偶漂亮或有人緣;但易驕傲、無禮(尤其子卯組合) |
| 花在時支 | 對事業有好的緣分;子女有人緣 |
| 花在月＋年(兩花相連) | 性慾較高 |
| 四柱全花 | 如果變格,反而不得人緣,並耽誤婚姻 |

- 圖 45 右頁附「命盤 13」實例(1951 年女命,正官格,地支兩卯兩戌多重合),圖 46 左頁附「命盤 14」實例(1960 年男命,七殺格,辰戌沖+半三合),皆為數花與標示的示範,細部數據從略。

## 四、四庫:辰、戌、丑、未(財庫——聚寶盆)(圖 47)

財庫主錢財的收藏與調度能力。依庫的數量論斷:

| 庫數 | 論斷 |
|---|---|
| 無庫 | 錢財守不住、散財童子、財來財去;自己很節儉,但錢不是自己的 |
| 一庫 | 很節儉;疼別人——節儉自己、慷慨別人 |
| 二庫 | 很能調度金錢、善理財;但庫逢沖則開銷大、難聚財 |
| 三庫 | 很會賺錢,但財容易分散、出手大方、四處投資;借錢不用還(借出去的錢不敢要回來) |
| 四庫 | 走極端:**身強**又走運時賺進天下財;**身弱**又不走運時散盡天下財(所謂「皇帝就是乞丐財」——賺盡天下財、也可能散盡天下財;非皇帝命格即乞丐命格) |

財庫互沖的補充:

- **辰戌沖**:自圓其說、自找台階、喜做老大、脾氣不好、理由多;財庫沖開、開銷大;好鬥、好訴訟。
- **丑未沖**:打破砂鍋問到底、好問、鑽牛角尖;過度自信、易口角、是非多、易賠錢;財庫沖開、開銷大;凡事多阻逆。

### 庫所在柱位的論斷(圖 47)

| 庫的位置 | 論斷 |
|---|---|
| 庫在年支 | 身強者較能得到父母的不動產;賺的錢較會放在父母處 |
| 庫在月支 | 節儉自己、慷慨別人;若天干為財星,則表「財入庫」 |
| 庫在日支 | 配偶帶庫、聚財有力;賺錢要繳回給配偶;配偶易堅持己見、管東管西;亦代表結婚後才較會存錢 |
| 庫在時支 | 會留不動產給子女、較疼子女;亦代表生了孩子之後較會存錢 |
