# 地支刑沖合害會

> 來源:《學八字,這本最好用》第四章(掃描圖 32–37,書頁 90–101)。
> 主旨:十二地支彼此之間的「刑、沖、合、害、會」關係,是從命盤直接論斷吉凶禍福的核心工具;三合、三會、六沖、六合、六害、三刑各有其代表意義。

## 一、地支三合(圖 32)

- 三個地支依「生、旺、庫」的位置構成一個五行局。
- **半合**:三合局中只出現其中任兩個地支,即成半合,力量較完整三合弱。

| 三合局 | 化局 | 生 | 旺 | 庫 | 性格特質(命帶此局者) |
|---|---|---|---|---|---|
| 申子辰 | 水局 | 申 | 子 | 辰 | 智慧多元、變化大、能臨時應變改變做法;但偏冷眼旁觀,易顯冷漠自私 |
| 巳酉丑 | 金局 | 巳 | 酉 | 丑 | 有指揮領導能力、講義氣;外表威嚴冷酷,帶肅殺之氣,易見血光;說話懂得修飾 |
| 寅午戌 | 火局 | 寅 | 午 | 戌 | 熱情、行動派、急性子;但常前熱後冷,三分鐘熱度 |
| 亥卯未 | 木局 | 亥 | 卯 | 未 | 心地軟、愛幻想、計畫一大堆;多不切實際,大部分難以實現 |

## 二、地支三會(圖 33)

- 三會是同一季節、同一方位的三支會成一方之氣,**力量大於一切**(強過三合、六合)。
- 不必三支全齊:命盤中有其中兩支,再加流年補齊一支,也能組合成三會。
- 影響:日主弱者遇三會其印比之氣反而有利;日主強者再會出旺氣則不利,該年會特別有事情發生。
  - 書中例:日干屬木,地支有寅卯辰(或其中二支加流年組成),身弱者有利、身強者不利。

| 三會 | 方位 | 五行 |
|---|---|---|
| 寅卯辰 | 東方 | 木 |
| 巳午未 | 南方 | 火 |
| 申酉戌 | 西方 | 金 |
| 亥子丑 | 北方 | 水 |

## 三、地支六沖(圖 33–34)

### 總論
- 沖=撞擊力,也是「播種」之象;帶沖的人很會做事、有執行力、有衝勁。
- 缺點:衝動、易與人意見不合而反目。
- **走運時逢沖則發;不走運時逢沖則墜落。**
- 太歲(流年)沖向命盤較嚴重;命盤沖向流年太歲則較輕。
- 「根」(地支)最忌沖;逢沖時要小心「財」的損失。

### 六沖逐條

| 六沖 | 論斷重點 |
|---|---|
| 子午沖 | 水火不容:情緒不穩、脾氣不好、個性極端,發瘋與腦神經衰弱的機率較高;但人緣好、異性緣佳,子午沖的人通常外貌漂亮 |
| 丑未沖 | 愛追根究底、打破砂鍋問到底、鑽牛角尖,主觀意識強烈;財庫被沖開,開銷大、易賠錢;較會與鄰居起口角;女命注意流產 |
| 寅申沖 | 勞碌命,忙碌閒不住;六親較無緣,一輩子靠自己(似傷官見官,一生大起大落);開車快、愛走大馬路,有車關、易生車禍 |
| 卯酉沖 | 桃花沖:做事俐落敏銳、有第六感、眼睛銳利;人緣好、異性緣佳,但心性不定 |
| 辰戌沖 | 辰戌屬現金,庫沖庫;沖到事業宮則投資易失敗;個性不服輸、自圓其說、自找台階、喜當老大、脾氣不好、理由多、把錯歸給別人;野心大、開銷大;須注意婚姻問題 |
| 巳亥沖 | 辯才無礙、口才佳、愛聊天,但常禍從口出;凡事追根究底;開車愛鑽小巷,有車關 |

## 四、地支六合(圖 34)

- 六合代表:有計畫能力、合得來、有收成、能守成、同心協力、好溝通。

| 六合 | 化氣 |
|---|---|
| 子丑 | 合土 |
| 寅亥 | 合木 |
| 卯戌 | 合火 |
| 辰酉 | 合金 |
| 巳申 | 合水 |
| 午未 | 合火 |

## 五、地支六害(圖 35)

### 總論
- 害=**分離**(人的生離死別)、變卦、聚少離多、同床異夢;快要收成時反而收不到。

### 六害逐條

| 六害 | 論斷重點 |
|---|---|
| 子未害 | 最嚴重的害,又稱「天地害」「南北害」:個性極端、易犯小人、常換工作;與人貌合神離、無話可說,卻又會要求對方 |
| 丑午害 | 耐性差、容易生氣,與人貌合神離 |
| 寅巳害 | 是非多、無恩情(人情)、易犯小人;冷眼旁觀的態度;屬「驛馬害」,辯才無礙;若離婚,也可能仍同住一個屋簷下 |
| 申亥害 | 是非多、無恩情、易犯小人;屬「驛馬害」;如「相見不如懷念」——見面就吵,不見又想念 |
| 卯辰害 | 殺傷力很大:易遭身邊親人所害、好友扯後腿;兄弟無緣、手足無助;你對他愈好反而愈害到自己,愈親近的人反駁力愈大 |
| 酉戌害 | 與卯辰害相似:容易被近親戲弄,弄得雞犬不寧、哭笑不得,離婚率高 |

## 六、地支之刑(圖 36–37)

### 總論
- 刑主**精神層面的壓力與挫折**,多屬內心的煎熬與鬱悶。

### 四種刑

| 刑 | 組成 | 核心意義 |
|---|---|---|
| 無禮之刑 | 子刑卯、卯刑子 | 自視清高、眼光高、講話不客氣沒禮貌,不隨便與人交談,看不順眼的人就不理;刑中最凶:不孝不悌、相妒不睦、剋損六親;婦人有此刑與翁姑不合、易損孕 |
| 恃勢之刑 | 寅刑巳、巳刑申、申刑寅 | 替人打江山做憨工,做到累死也沒人感激;有十分力留三分才能收尾;你嫌人、人也嫌你;事情做得成,但只要一小點沒做好就被修理;性情冷酷薄義,易遭陷害惡事;女子有此刑易損孕 |
| 無恩之刑 | 丑刑戌、戌刑未、未刑丑 | 過度自信、過於猛進而易遭挫折,缺乏惻隱之心;剛毅但易罹災;婦人有此刑易孤獨 |
| 自刑之刑 | 辰辰、午午、酉酉、亥亥 | 心裡的鬱悶找不到人說、有話說不出口,一點一點累積;明知不可為而為之,常拿石頭砸自己的腳,易想不開、自卑(亥月生者尤甚) |

- 恃勢之刑判讀技巧:看兩支各落在哪個宮位、各代表「誰」,即可推論誰為誰付出卻不被感恩。書例「寅刑巳」:巳做到流汗,還被寅嫌到流口水,寅不懂感恩。
- 無恩之刑書例「丑刑戌」:丑對戌太有信心,自認戌能幫自己搞定一切,結果往往事與願違。

### 自刑逐支(圖 37)

自刑輕重:**亥最重(尤其在月令)> 酉、午 > 辰(最輕)**。

| 自刑 | 特質 |
|---|---|
| 辰刑辰 | 固執、有原則,不喜被言語刺激;喜獨立當老大;懷才不遇,做事有頭無尾;鬱悶型 |
| 午刑午 | 好勝心強,不喜被人左右,吃軟不吃硬(愛聽好話、被拍馬屁);個性極端、沒耐性、健忘 |
| 酉刑酉 | 講義氣、雞婆熱心;遇懶散或不講理的人會生氣,乾脆不說;熱情過頭轉成憂鬱,鬱悶型,易有自殺傾向 |
| 亥刑亥 | 聰明有智慧、明理;但有事悶著不說,鬱悶累積,易有自殺傾向 |

- 圖 37 左頁附「命盤 8」實例(1951 年女命,正官格),盤中戌卯、戌卯多重合刑,用以示範自刑與合的實際標示;命盤細部數據此處從略。
