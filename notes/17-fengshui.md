# 陽宅開運

> 出處:《學八字,這本最好用》第九章「運用簡易的陽宅開運法來制煞解厄」(掃描圖 104–106,原書 p.234–239)。
> 三大主題:制煞防意外血光、布局旺財、淨化宅氣。

## 一、鎮宅制煞,防止意外血光(第一節)

### 羅盤制煞法

若由命盤得知當年會有意外或血光之事,可用羅盤預先制煞:

| 場所 | 作法 |
|------|------|
| 車上 | 掛**小羅盤**,防「車關」發生 |
| 房子 | 用**大羅盤**掛在客廳,或掛在流年「五黃煞」、「二黑土」的方位上 |

俗話說:預防重於治療。

### 粗鹽避厄法

- **觀念(PS)**:意外常被歸咎於「不小心」,但很多時候是自己很小心、別人不小心來害到我們——這叫「壞運」。防止運壞,要做功課。
- **行車平安**:平常騎車或開車者,約一個月一次在車輪胎前後揮灑粗鹽,讓車子保持乾淨磁場,厄運不靠近、行車一路平安。
- **居家/辦公/生意場所撒淨**:從屋中的**東北角**開始,**順時鐘**方向撒粗鹽(數量不用太多),約半年撒淨一次,可讓人感覺事事順利、財源廣進。
- 為什麼從東北角開始?因為**東北—西南連成的直線就是「鬼門線」**,所以撒淨從東北邊開始。

## 二、陽宅布局旺財(第二節)

### 客廳財位擺設

人生改運:除了認真賺錢,還要布局財位。老師們建議在**客廳財位**擺放以下開運物,即可催旺家中財運,使工作事業愈來愈順、財源滾滾:

- 開運納財聚寶盆
- 綠色圓葉盆栽
- 鎮宅催財貔貅
- 鎮宅催財麒麟

**禁忌(PS)**:財位上千萬不要擺垃圾桶、無門鞋櫃,以免污穢財神,財神不來照顧。

### 財位怎麼找?——房子財位及旺方速查表

作法:人站在**客廳正中心**,用羅盤或指北針定位,找出**大門**所在方位(門位),再查下表得財方與旺方:

| 門位 | 財方 | 旺方 |
|------|------|------|
| 南 | 東南、北 | 東 |
| 西南 | 東北 | 西北、西 |
| 西 | 西北 | 西南、東北 |
| 西北 | 西 | 西南、東北 |
| 北 | 東、南 | 東南 |
| 東北 | 西南 | 西北、西 |
| 東 | 東南、北 | 南 |
| 東南 | 東、南 | 北 |

### 各房間的檢查與擺設(福地福人居)

如果現在住的房子讓你感覺不順暢,先逐間檢查:

1. **客廳**(代表男主人的顏面):是否整齊、乾淨?
2. **廚房**(代表女主人的財庫及脾氣):是否整潔、乾淨?可種一棵綠色盆栽來美化及釋放芬多精,也可用門簾擋煞氣。
3. **臥房**:床是否有避開門、鏡子或樑柱?
4. **廁所**:所有定位都定位了嗎?

另可用**粗鹽**吸收廢氣。檢查各房間都很乾淨、清爽、空氣好,再加上陽宅布局催旺開運,效果會很好;反之則改善效果不佳。

## 三、淨化陽宅,一切平安順利(第三節)

### 磁場觀念

- 「磁場」好,一切平安順利;磁場不佳,則百事不順、身體欠安、百病隨之而來。
- 建議選用以下物品,讓住宅通氣、生活在充滿溫馨清淨的環境:
  - 開運薰香爐
  - 粗鹽開運鹽晶
- 據經驗:「磁場」經淨化後配合禪定觀想,一段時日後就會顯得身體健康、事事順利、財源增進——那就表示家中磁場與本身磁場相互融合了。
- PS:磁場兩個字常聽到,但看不到;要如何使用可請教老師。

### 老祖宗的粗鹽淨宅法

除上述物品外,也可用傳統粗鹽淨化法(堆鹽皆用杯子裝,每處約三十公克):

| 位置 | 作法 |
|------|------|
| 房子四周角落 | 各放 30 公克粗鹽 |
| 廁所內 | 放 30 公克粗鹽 |
| 廚房 | 放 30 公克粗鹽 |
| 大門入口處 | 放 30 公克粗鹽 |

- 以上方法可改善居家不好的宅氣,效果相當好。
- **記得一個月換一次粗鹽**,換下的粗鹽灑到外面的土地上讓它歸回大地。
