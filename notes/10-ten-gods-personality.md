# 十神論斷個性及六親對待

> 資料來源:《學八字,這本最好用》第五章第二節「由十神論斷個性及六親對待」(掃描頁 53–68,書頁 132–162)。

## 比肩

**核心特性:「獨立自主、重朋友」。**

### 個性優點
- 自尊心與自信心強烈,有自知之明,凡事量力而為,不貪非分之想。
- 樂觀進取、意志堅定,凡決定之事必堅持到底,不輕易改變、不懼不畏。
- 獨立自主、言而有信,對朋友感情真摯。
- 不輕易向他人低頭,一心渴望與他人並駕齊驅,堅守崗位、努力工作達成目標。

### 個性缺點
- 比肩太旺為忌則頑固不通、剛愎自用;凡事先想到自己,不考慮他人立場,缺乏容人雅量。
- 個性剛強,不易與人溝通,難與人打成一片;交友雖廣卻難得知己。
- 表面柔順、內心剛強,縱使口頭稱讚別人,心裡卻極為不服。
- 具抗上之心,較得不到上司的提拔與器重;遇困難時也不易獲得外援。

### 身強／身弱差異
- **身強**:見上方「身強者通性」——獨立孤獨、與人爭取計較、必有競爭者、比較會失戀、好勝不服輸。
- **身弱**:為人我行我素、不喜束縛,任何人都無法改變他;理財上儘量不要跟會(民間互助會)、不要合夥,錢財易被掠奪。

### 通則:比肩+劫財同時出現(比劫過重)
- 個性陰沉,不喜表達心中意願;一旦恨上某人,永不與對方講話,難以翻身。
- 善於詢問,但對自己沒信心,常改變決定。
- 錢不要借人、不要跟會、不要合夥——易被劫走,常掉東西。
- 外表樂觀、內心想不開,甚至有輕生念頭;憑直覺做事、耳根軟、心神不專易分神,常往壞處想。
- 人情包袱重(太重人情),要提防落入紅塵環境;一生感情困擾,剪不斷理還亂。

| 年柱 | 小時候跟父母花很多錢,或一輩子錢財被父母、長上劫走;常聚一群趕不走卻沒幫助的朋友,易與朋友有錢財糾紛,好事難成,易走入不好的環境。個性穩健平和;小時候好玩、不喜讀書、好交朋友,有錢與人平分共用;小時候家庭經濟易陷入危機(父母缺錢或為自己看病花錢),一出生家境不富裕;從小優柔寡斷但乖巧懂事,常身兼數職分擔家計。 |

## 劫財

**核心特性:與比肩同為「獨立自主、重朋友」,但更外向、更具行動力與雙重性格。**

### 個性優點
- 性格獨特突出、獨樹一格;外交能力與口才優異,特別善於在社交場合製造氣氛、控制局勢博取好感。
- 心思敏捷迅速、善見風轉舵,能因應他人需求與環境變化,應變能力一流。
- 行動力強、迎合時尚;謀事積極、勇往直前、冒險犯難、鬥志昂揚。

### 個性缺點
- 具雙重性格:外表與內心經常自我衝突、自相矛盾、表裡不一,朋友來得快去得快。
- 性情陰晴不定、忽冷忽熱、難以捉摸;有時薄己利他,有時又嫉妒他人成就。
- 野心過大、求功心切,常不經思考貿然行動而一敗塗地;孤注一擲弄得無法收拾。
- 做事橫衝直撞、有勇無謀,動輒想用武力解決、不計後果。
- 不善處理金錢,對朋友過分慷慨,易因兄弟朋友之事與配偶起爭執;對外面的女性體貼多情,對妻子卻顯得冷漠。

### 通則(男女命差異)
1. **男命帶劫財**:容易失戀,婚前交女友易有人爭;要做什麼事前最好不要先說,否則易破功;對老婆較沒安全感。
2. **女命劫財過重**:劫財落在任一柱,戀愛時另一半身邊會出現追求者,失戀機會很大,且是刻骨銘心的失戀(一個劫財約失戀 2~3 次);感情上易當老二(第三者)。

### 四柱位置的六親對待
| 柱位 | 論斷 |
|------|------|
| 年柱 | 小時候跟父母花很多錢,或錢財一輩子被父母、長上劫走;小時候家裡會發生事故,會分攤家庭經濟;容易欣賞成熟的異性(早熟);父母是他一生最大的掛記,時常記憶小時候之事;出生時正逢家中最困窘的時候;從小體恤懂事、好玩、好交朋友。 |
| 月柱 | 兄弟姊妹與朋友排第一,配偶反而排最後,常遭埋怨;感情易被朋友分割(男女朋友被搶);早年生活非常辛苦;易因朋友而失財——付出多、替人擔憂,別人卻不一定幫忙;會為朋友、平輩、兄弟姊妹而失財。 |
| 日柱 | 1. 配偶易破財;夫妻常有感情困擾之事。2. 有錢時兄弟朋友多、沒錢無人過問;錢有借無還、容易被欠錢又不好意思要回來;結婚後開始走拮据的人生。 |
| 時柱 | 年老時生病吃藥效果不好;與子女互不喜同住。事業被夥伴及子女劫財;很疼兒女、有求必應;晚年財務不好,借人錢不敢要回;越老越自卑(自尊心強之故),大都不跟子女同住;答應的事常反悔改變;子女對外求生能力較弱。 |

## 食神

**核心特性:「斯文、歡喜心」——按條理走,有口福且口感特好,總能找到好吃的食物;有氣質、纖細柔美。**

### 個性優點
- 氣質清高、溫文儒雅、性格開朗、精神和暢;聰明細膩、通情達理、寬容厚道、和平善良、不與人爭、驕而不傲。
- 感情豐富、優游自足、思想清新脫俗,喜愛美好事物;表達生動流暢但不誇張,雖出風頭但含蓄不露鋒芒。
- 重視情調,調和精神與物質的均衡發展;對文學、藝術、歌舞有偏好且感受力敏銳;精於飲食之道,一生衣食豐厚。
- 與人相處含蓄保守、溫厚有禮、不任性叛逆;謀事專一,習一技而至精純,能深入而成專家;樂於付出而不強求回報。

### 個性缺點
- 思想清高,易自命不凡;理想過高以致與現實脫節,幻想過度而生空虛寂寞之感。
- 思考旺盛容易疲勞、常感體力不支;喜歡優游、忽視現實,易失去奮發進取的精神;易沉迷於遊樂。
- 喜動腦筋,易腦神經衰弱或頭疼,有志難伸之感。

### 身強／身弱差異
1. **身弱**:才華不顯(無法用來賺錢),體力較差;學東西不穩定,樣樣都想學卻樣樣不精,流於雕蟲小技。(若成從兒格或從勢格則反之)
2. **身強**:真有才華、較能賺錢;較會把讀書所學應用於生財之道,能發揮實才。(若成從格則反之)

### 通則
- **食神＋傷官多**:常因自負而斷送財根;對自己過分自信,難聽他人意見。

### 四柱位置的六親對待
| 柱位 | 論斷 |
|------|------|
| 年柱 | 小時候很快樂;會思考、頭腦好、讀書學習靜得下來;聰明伶俐、學習力強、善體人意;小時候體衰力弱,是父母最操心的小孩;父母靠胼手胝足奮鬥、不靠祖業;一出生就有得吃、家境不錯;工作上文武皆可勝任。 |
| 月柱 | 會研究及發明東西;交的朋友都不錯,但朋友中也易引來官司訴訟(逢官時);不喜用祖德,想靠自己雙手獨立賺錢;適合文學及翻譯人員,能運用頭腦賺錢,較有食祿。 |
| 日柱 | 1. 日支藏干食神:配偶有才華、較喜樂、易發福。2. 配偶性格敦厚寬宏,夫妻相處互不拘束;配偶身材較豐厚。 |
| 時柱 | 長壽;老年快樂、肚量大;晚年較會吃素;女命注意流產(逢偏印 P 時易剋損子息或生產不順);自身行事緩慢、企畫力強但實踐力弱;口才佳、感情重實質生活;配偶較風流、注重性生活協調;早出社會、容易付出;喜歡兒孫滿堂;煩惱約只停留五分鐘。男命食神為祿壽,女命為子息。 |

## 傷官

**核心特性:「易被感動、男英俊、女俏麗」——凡事惜情、重感情、過分感情用事。**

### 個性優點
- 博學多能、多才多藝、活潑善辯、表達流暢;聰明足智、機靈敏銳、深謀遠慮、領悟優異。
- 有不斷超越他人的慾望,學習能力強、理想高遠、滿腔抱負、活力充沛、鬥志高昂,重視他人對自己的肯定。
- 創意豐富、口才流利、表情豐富:適合演藝、歌唱、舞蹈、廣播、新聞報導、節目主持等靠口才的工作;對藝術美學感受敏銳、頗具靈性,易成英雄人物,大都相貌清秀或博學多能;亦適合畫家、自由業、古董或精密技術方面發展。

### 個性缺點
- 領悟力強反而興趣廣泛、博而不精;不能量力而為,常做出超越本身能力之事而招致失敗。
- 恃才傲物、任性乖張、驕傲自大、一意孤行、無法接納忠言;厭惡禮俗拘束。
- 易傾向狂妄、蔑視法令;為達目的不擇手段、以私害公、傷人自尊、愛出風頭,招來許多誤會與禍患。
- 好勝逞強、乖戾不馴;財多則貪得無饜、好管閒事;做事容易情緒化。
- 女性宜注意修心養性,以免違反倫常、觸犯法律;男性宜盡量克制私慾。

### 通則
1. **偏印＋傷官同現**:獨斷又缺乏自信,常做了後悔又不願承認;會常恍惚不知在想什麼;才華不穩定;重邏輯、表現慾強、理解力強;多學不專、急性、喜自由、易感動、易受外在事物影響。
2. **傷官多(兩個以上)**:血光、破財、官司、車關,一生災難不斷;驕傲自負,常因自負斷送財根;看不得他人比自己好;做事反反覆覆而不自知,破壞自己格局。
3. **比肩、傷官過重之女子**:較容易從事八大行業,也較多愁善感;環境較差、處處須自己眼明手快打點;把自己的環境看得較悲哀;個性不願落人於後、比較敢衝;易有精神疾病現象,或導致配偶送精神病院。
4. **傷官多(兩個以上)補充**:剛愎自用;學東西很快、學了就走,學習力強,相對看不起學不會的人;男性一生難有成就;女性一生愛做、愛唸、愛挑剔,又是直腸子的人。
5. **帶食神＋傷官者**:做事宜默默地做、不要強出頭、等人發現;「一體兩面」拿捏得很好。

## 正財

**核心特性(身強用):「愛玩、有歡喜心、喜愛熱鬧、有異性緣、疼老婆、節儉、做事固定、賺錢機會多、有家庭責任、滿足慾低、好溝通、閒不住、缺恆心」。**
喜固定資產、定期存款、房子、穩定股票、穩定工作。男命喜成熟的女人;菩薩心腸,自己有時也希望旁邊的人擁有。

### 個性優點
- 為人節儉、重視信用、安分守己、量力而為、不貪非分、正直不阿、嫉惡如仇、不喜投機。
- 刻苦耐勞、任勞任怨、辛勤努力、按部就班、知足常樂、腳踏實地、保守中庸。
- 珍惜金錢、點滴致富;家庭觀念濃厚、對妻兒盡責、疼老婆;眼見為信,不信邪魔歪道;不標新立異、不惹是非、不出風頭。

### 個性缺點
- 重視物質追求、缺乏精神調節,以致精神空虛——所謂「家富心窮」。
- 貪圖安逸、享受現成;對女性不吝其財,但對家族親友常斤斤計較,讓人覺得薄情寡義。
- 做事過於謹慎,缺乏勇氣魄力去追求變化與突破,生平事業較平淡無奇;虎頭蛇尾、半途而廢,缺乏轟轟烈烈的表現與成就。
- 單調刻板、憨厚保守、進取不足、權變不足;因小失大;欠缺責任感與榮譽感時容易流於怠惰。

### 通則
1. **男命有兩個正財**:有結兩次婚的機會(因離婚或喪偶)。八字無財不疼老婆,就是疼也表達不出來。
2. **天干有正財、偏財而地支無庫**:一生財來財去,表面看似有錢,內在未必有。
3. **正財多**:好溝通、會講好話,很有異性緣,有歡喜面。

## 全篇學習要點

1. 論斷順序:先判身強身弱 → 看哪個十神多、落在哪一柱 → 套用該十神特性與柱位六親對待。

4. 「某十神完全沒有」也有意義:八字無財不疼老婆、無官殺說話不威、無印星沒主見沒安全感。
