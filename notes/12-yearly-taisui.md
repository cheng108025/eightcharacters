# 斷流年太歲之刑沖合害吉凶

> 出處:《學八字,這本最好用》第七章(掃描頁 84–91)

## 總論:太歲與本命盤的四種關係

- 「太歲」即流年的干支,論斷時主要取**流年地支**,與本命盤四柱地支(年支、月支、日支、時支)逐一比對。
- 比對後可能形成四種關係(書中的四項診斷條件):
  1. **沖**(六沖)
  2. **刑**(三刑、自刑)
  3. **合**(六合、三合)
  4. **害**(六害)
- 流年地支與哪一柱地支產生刑沖合害,吉凶現象就應在該柱所代表的人事領域:
  - **年柱**:長輩、上司、祖上、外在環境
  - **月柱**:兄弟平輩、內心世界、家庭(月支亦為提綱)
  - **日柱**:本人與配偶(夫妻宮)
  - **時柱**:子女、部屬、事業機運
- 書中以命盤 23~27 為例:如流年地支戌與本命月支卯為卯戌合(太歲來合)、流年申沖本命年支寅(太歲來沖)、流年申害本命時支亥(太歲來害)、流年申刑本命月支巳與日支寅(太歲來刑)、流年酉與本命時支酉為自刑等,依所在之柱論斷該年現象。

## 太歲來「合」(對應在流年論斷上)

合有「絆住、動彈不得」之意,合到哪一柱,該領域的事今年就施展不開。

| 太歲合入之柱 | 流年現象 |
|---|---|
| 年柱 | 今年會為長輩、上司、廠商之間的事而煩心,自己較施展不開;有投資當老闆的可能 |
| 月柱 | 今年凡事自己會鬱悶、較沒動力,無法突破、踏不出去;內心想了很多計畫,卻沒付諸行動 |
| 日柱 | 今年的計畫會因配偶的關係而無法拓展;因合入異性宮,心裡衝動、想歸想卻未必有行動,耳根軟;未婚者會想結婚 |
| 時柱 | 對事業的看法比較沒有衝勁;家庭方面會為了小孩的事煩憂,而無法拓展事業;或自己想有新事業,是個顧家的人;想創業者,今年易有人找你合夥做事業 |

### 各種「合」的組合性質(命盤如果呈現)

| 合的組合 | 性質 |
|---|---|
| 子丑合 | 夫妻間好溝通、有話講,較顧家 |
| 卯戌合 | 比較愛面子、注重外表,顧家,外強內柔 |
| 寅亥合 | 先天磁場上比較重視倫理道德 |
| 辰酉合 | 先天磁場上比較重義氣,但比較沒有定性 |
| 午未合 | 先天磁場上比較積極,天生脾氣不好,做事重感覺 |
| 巳申合 | 很多事會聰明反被聰明誤;此組合屬「無恩之刑」 |
| 申子辰三合水 | 思想變化快、很聰明,但較冷漠 |
| 寅午戌三合火 | 看來做事效率高、執行力好,熱情有禮 |
| 亥卯未三合木 | 較會有不切實際的想法,較有夢想(白日夢) |
| 巳酉丑三合金 | 為人較會強出頭,較會有血光之災,且會包裝自己 |

## 太歲來「沖」(對應在流年論斷上)

沖為對立、衝突、變動之象,沖到哪一柱,該領域今年易起衝突或變化。

| 太歲沖到之柱 | 流年現象 |
|---|---|
| 年柱 | 今年容易與長上、上司起衝突,有意見不合、相處不來或不懂得忍讓;同時需注意身體方面的狀況 |
| 月柱 | 今年心性上會產生變化(會想動),會主動做改變——以前不敢動的,今年就敢;同時與平輩、兄弟之間會有衝突 |
| 日柱 | 未婚:會有人追(紅鸞星動);已婚:夫妻易起衝突、吵架或婚變 |
| 時柱 | 事業會有大的起伏(有人爭取職位),內心想創業;和小孩、部屬較有衝突或不合的現象;行動力佳、適合外務,東奔西跑,但對人付出人家不太會感激 |

### 各組沖的性質(六沖分為馬、花、庫三類)

| 沖的組合 | 類別 | 流年機運與性質 |
|---|---|---|
| 寅申沖 | 馬 | 今年須注意車關、不要開快車;適合外務、東奔西跑;對人付出,人家不太會感激 |
| 巳亥沖 | 馬 | 會有車關;口才好、辯才無礙,比較會得理不饒人;易抄近路、求速度(會鑽);適合外務 |
| 子午沖 | 花 | 個性極端、反覆不定、不堪諷刺;桃花動、人緣好;生氣時會抓狂,有感情困擾 |
| 卯酉沖 | 花 | 第六感強(有時會看到第三度空間);目色好,凡事須先溝通好才做;東西常移動、桃花動、人緣好;長輩緣好,較會衡量人、憑直覺做事 |
| 辰戌沖 | 庫 | 個性不好、脾氣難以控制,較會自圓其說;「庫沖破,財就守不住」,有運賺、無運賠;辯才無礙、好辯,身體較會不好 |
| 丑未沖 | 庫 | 凡事追根究底、打破沙鍋問到底,較會查行蹤;庫沖破,錢財守不住,小財不斷流失 |

> 「馬」主驛動、車關;「花」主桃花、人緣、感情;「庫」主財庫,沖破則財難守。

## 太歲來「害」(對應在流年論斷上)

害主暗傷、小人、分離,害到哪一柱,該領域今年多煩憂波折。

| 太歲害到之柱 | 流年現象 |
|---|---|
| 年柱 | 今年常為長上的事或其身體煩憂;十六歲前則是長上為你煩憂;或會被上司炒魷魚,與長上有心結、代溝,易換上司 |
| 月柱 | 今年心情特別亂、有苦說不出口、很鬱悶;計畫會有變卦、無法完成;有搬家的可能 |
| 日柱 | 未婚:今年可能會分手;已婚:會想離婚,或會產生婚變或分居 |
| 時柱 | 今年事業會有波折、變動或離職;和小孩會分離,或小孩會出事且不聽話 |

### 各組害的性質(六害)

| 害的組合 | 性質 |
|---|---|
| 子未害 | 個性極端,容易犯小人,親子之間較早分離 |
| 丑午害 | 耐性差、脾氣不好,容易生氣 |
| 寅巳害 | 屬無恩之刑,是非多、易犯小人,在家裡待不住 |
| 卯辰害 | 容易被扯後腿,兄弟姊妹無助,易遭親人相害 |
| 酉戌害 | 容易被好朋友扯後腿,易遭親人相害,弄得雞犬不寧 |
| 申亥害 | 是非多、容易有小人,挫折感重,受不了刺激 |

## 太歲來「刑」(對應在流年論斷上)

刑主刑剋、心理煎熬、相互虧欠之感,刑到哪一柱,該領域今年多莫名的煩悶。

| 太歲刑到之柱 | 流年現象 |
|---|---|
| 年柱 | 今年本身與長輩、長官之間有一種莫名其妙的感覺,總是很煩 |
| 月柱 | 今年配偶會因長輩的行為或言語而鬱悶很久 |
| 日柱 | 今年長上與子孫、上司與部屬之間有一種相互虧欠、「恨鐵不成鋼」的感覺,真的需要好好溝通 |
| 時柱 | 今年本人與子女或部屬間的認同度不夠,因此會產生敵對的心態 |

### 三刑的種類

| 刑的種類 | 組合 | 性質 |
|---|---|---|
| 無禮之刑 | 子卯 | 自命清高(眼光高),說話直接、較沒禮貌(沒大沒小),自以為氣質好 |
| 無恩之刑 | 寅巳、巳申、申寅 | 無人賞識,做事易被人嫌棄;替人打天下、任勞任怨,愛恨交加 |
| 恃勢之刑 | 丑戌、戌未、未丑 | 太自信、自負欠考慮,憑感覺投資;其中未丑刑:虧錢機率高,錢看不見卻不斷流失 |

## 自刑(辰、午、酉、亥)

- 自刑限於**辰、午、酉、亥**四支:只要本命地支中有一柱與太歲地支相同(辰見辰、午見午、酉見酉、亥見亥),即構成自刑條件。
- 自刑的共同現象:對自己不滿意、易憂愁;明知不可為而為之;自尊心強、自尋煩惱、容易鬱悶、有話不說;若自刑在月柱,更是難以溝通。

| 自刑組合 | 性質 |
|---|---|
| 辰辰 | 要別人聽自己的,但常事與願違而自我鬱悶;請用逆向溝通(先拍馬屁)的方式才能得到認同;做事直接,但有遠觀 |
| 午午 | 常常事與願違;在決定一件事之前,用逆向溝通方式(先拍馬屁)成功機率會較高;以禮相待則好溝通 |
| 酉酉 | 常常想幫助別人,卻得不到認同,真是鬱悶;常幫倒忙,但為人熱心、多慮 |
| 亥亥 | 較會無理取鬧、歇斯底里、悲觀;總是想「我比別人認真,為什麼比別人歹命」;如又逢流年來害月柱,容易有輕生念頭;喜用頭腦,卻多愁善煩、易想太多 |
