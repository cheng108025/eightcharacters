# 由命盤地支看運勢及六親對待

> 來源:《學八字,這本最好用》第四章第二節(掃描圖 38–43,書頁 102–113)。
> 主旨:命盤四柱(年、月、日、時)地支之間出現合、沖、害、刑時,依其所在宮位可直接論斷本人與長上、配偶、子女、事業之間的對待關係。
> 宮位對應:年=長上/祖上,月=父母/長輩/上司,日=自己與配偶,時=子女/部屬/事業晚年。

## 一、命盤有六合或三合時(圖 38)

書中以表格列出八種情況(表格以 * 標示合的柱位;掃描星號位置略模糊,以下依文意對應):

| # | 合的位置 | 論斷 |
|---|---|---|
| 1 | 年支＋月支合 | 容易得到長上幫助,與長輩、上司溝通良好、有長上緣,易被提攜器重;孝順、有老人緣,凡事較以長上意見為主 |
| 2 | 月支＋日支合 | 配偶與長上相處佳、容易獲長上幫助;配偶多是長上介紹或認可的;長上會幫忙你的事業,也會以金錢支助 |
| 3 | 日支＋時支合 | 長上會無條件幫你帶小孩;人緣好、做事有自己的主見,容易滿足現狀、較有安全感,但不易透露內心 |
| 4 | 年支＋日支合 | 夫妻相處佳、在乎另一半;創業時能堅持;但若同時逢害,則一切無所適從 |
| 5 | 月支＋時支合 | 在意孩子、事業、員工、晚輩;人際關係好、能顧全大局(公務人員居多,事業做得比較久) |
| 6 | 年支＋時支合 | 配偶會支持、相助你的事業;只要創業就會堅守做下去 |
| 7 | 天干合 | 與地支合意義相通,主和諧、互助 |
| 8 | 天干合、地支刑 | 身體不好(四柱中有一柱被太歲刑時亦同論) |

### 各組六合在對待上的性質(圖 38 左頁)

| 六合 | 對待特質 |
|---|---|
| 子丑合 | 夫妻間好溝通、有話講,較顧家 |
| 卯戌合 | 比較愛面子、注重外表;顧家,外強內柔 |
| 寅亥合 | 先天磁場上較重視倫理道德 |
| 辰酉合 | 先天磁場上比較重義氣,但較沒有定性 |
| 午未合 | 先天磁場上比較重情意,天生脾氣不好,做事重感覺 |
| 巳申合 | 有很多事會聰明反被聰明誤;事屬無恩之刑(巳申既合又刑) |
| 申子辰合水 | 思想變化很快、很聰明,但較冷漠 |
| 寅午戌合火 | 看來做事效率高、執行力好,熱情有禮 |
| 亥卯未合木 | 較會有不切實際的想法、有夢想(白日夢) |
| 巳酉丑合金 | 為人較會強出頭,較會有血光之災,且會包裝自己 |

## 二、命盤有六沖時(圖 39–40)

六種情況(依沖所在柱位):

| # | 沖的位置 | 論斷 |
|---|---|---|
| 1 | 年支＋月支沖 | 和長上、上司容易意見不合、起衝突;個性較衝動,多早離家 |
| 2 | 月支＋日支沖 | 配偶和長上容易意見不合;長上、上司常是敦促你創業的人,且放不下、一定會干預 |
| 3 | 日支＋時支沖 | 長上對你的事業有意見、祖孫易不合;外表像靜不下來的人,子女像全身裝了馬達,好動不停息 |
| 4 | 年支＋日支沖 | 夫妻間常鬥嘴、無法溝通(男命男起因、女命女起因),所謂床頭吵床尾和 |
| 5 | 月支＋時支沖 | 對事業積極、有衝勁、活力全開;但容易對小孩或部屬有意見、不合 |
| 6 | 年支＋時支沖 | 配偶對你的事業易有意見(無助),和小孩溝通不良;家中之事都是配偶在敦促、打罵小孩 |

- 圖 39 右頁附「命盤 9」實例(1977 年女命,傷官格):地支巳亥沖、巳亥沖並列,示範沖的標示;命盤細部數據從略。
- 圖 40 右頁補充各沖在個人特質上的論斷(與筆記 05 六沖條目相同意旨):
  - 寅申(馬):注意車關、不開快車,東奔西跑,對人付出但人家不太會感激。
  - 巳亥(馬):口才好、辯才無礙,較會得理不饒人、易抄近路、求速度。
  - 子午(花):個性極端、反覆不定、不堪諷刺,感情困擾多。
  - 卯酉(花):第六感強(有時會看到第三度空間)、目色好;移動、桃花動、人緣好;長輩緣佳,較會衡量人、憑直覺做事,須先溝通好才做,東西常移動、桃花動、人緣好。
  - 辰戌(庫):個性不好、脾氣難以控制、較會自圓其說、庫沖破,財就守不住,有運賺、無運賠,辯才無礙,身體較會不好,好辯。
  - 丑未(庫):追根究底、打破砂鍋問到底,較會查行蹤、庫沖破、錢財守不住,小財不斷流失。
- 圖 40 左頁附「命盤 10」實例(1962 年男命,正財格):地支戌酉相害、半三合並見,示範標示;細部數據從略。

## 三、命盤有六害時(圖 41)

六種情況:

| # | 害的位置 | 論斷 |
|---|---|---|
| 1 | 年支＋月支害 | 自己和長上、上司不易溝通,有心結、代溝;不喜攀緣上流社會 |
| 2 | 月支＋日支害 | 配偶(是你往上成長路上的石頭)和長上無緣;長上對配偶有意見;想結婚就別在婚前讓父母參與意見 |
| 3 | 日支＋時支害 | 長上不認同你的事業;子女和長上無緣,長上不會幫忙帶小孩 |
| 4 | 年支＋日支害 | 不會在意配偶(丁壬合官亦同論):夫妻心結多、無法相處太久(聚少離多、同床異夢、易婚變或離婚);對初創事業不敢嘗試;未婚時談得投契,婚後相對無言 |
| 5 | 月支＋時支害 | 與小孩不易溝通;經常換工作、工作無法專心,常流於敷衍,較無責任感 |
| 6 | 年支＋時支害 | 配偶不認同你的人際關係,甚至不喜歡、不支持你的事業;和小孩難溝通,小孩難養或不能生育;性生活不美滿 |

- 特例:**配偶宮申亥害**——夫妻很恩愛,但其中一方會身體不好。

| 六害 | 歌訣 | 白話論斷 |
|---|---|---|
| 子未害 | 羊鼠相逢一旦休 | 個性極端、易犯小人;親子之間較早分離 |
| 丑午害 | 從來白馬怕青牛 | 耐性差、脾氣不好、容易生氣 |
| 寅巳害 | 蛇逢猛虎似箭投 | 無恩之刑性質,是非多、易犯小人;在家裡待不住 |
| 卯辰害 | 玉兔見龍雲裡去 | 容易被扯後腿;兄弟姊妹無助、易遭親人相害 |
| 酉戌害 | 金雞玉犬淚雙流 | 易被好友扯後腿、遭親人相害,弄得雞犬不寧,離婚率高 |
| 申亥害 | 豬見猿猴似箭投 | 是非多、易有小人;挫折感重、受不了刺激 |

## 四、命盤有刑時(圖 42–43)

六種情況:

| # | 刑的位置 | 論斷 |
|---|---|---|
| 1 | 年支＋月支刑 | 本身與長輩、長官之間有一種莫名其妙的感覺,總是很煩 |
| 2 | 月支＋日支刑 | 配偶會因長輩的行為或言語而鬱悶很久 |
| 3 | 日支＋時支刑 | 長上與子孫、上司與部屬之間相互虧欠、恨鐵不成鋼,需要好好溝通 |
| 4 | 年支＋日支刑 | 夫妻相處總是很不順心,也常惹對方生氣 |
| 5 | 月支＋時支刑 | 本人與子女或部屬間的認同度不夠,因此產生一種敵對心態 |
| 6 | 年支＋時支刑 | 配偶與子女及事業上的看法不一致,而很鬱悶 |

### 各類刑的補充論斷(圖 42–43)

- **無禮之刑(子卯)**:自命清高(眼光高)、說話直接、較沒禮貌(沒大沒小),自以為氣質好。
- **無恩之刑(寅巳、巳申、申寅)**:無人賞識,做事易被人嫌棄;替人打天下、任勞任怨,愛恨交加。
- **恃勢之刑(丑戌、戌未、未丑)**:太自信、自負欠考慮,憑感覺投資。其中「未丑」:虧錢機率高,錢看不見卻不斷流失。
- **自刑(辰、午、酉、亥)**:
  1. 對自己不滿意、易憂愁;明知不可為而為之;自尊心強、自尋煩惱、容易鬱悶、有話不說;自刑落在**月柱**時最難溝通。
  2. 逐支:
     - **辰**:要別人聽自己,但常事與願違而自鬱;對他請用「逆向溝通」(先拍馬屁)才能得到認同;做事直接,但有遠見。
     - **午**:常事與願違;決定事情前先用逆向溝通(先拍馬屁)成功率較高;以禮相待則好溝通。
     - **酉**:常想幫助別人卻得不到認同,覺得鬱悶;常幫倒忙,卻依然熱心。
     - **亥**:較會無理取鬧、歇斯底里、悲觀,總覺得「我比別人認真,為什麼比別人歹命」;若再逢流年來害月柱,容易輕生;喜用頭腦,卻多愁多煩事。

- 圖 42 右頁附「命盤 11」實例(1963 年男命,正財格):地支子卯刑、半三合並見;圖 43 左頁附「命盤 12」實例(1981 年男命,偏財格):亥寅巳刑害合交錯。皆為宮位刑的示範,細部數據從略。
