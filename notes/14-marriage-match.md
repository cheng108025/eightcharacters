# 診斷兩人姻緣吉凶(合婚)

> 出處:《學八字,這本最好用》第七章第三、五節與合婚判斷法(掃描圖 93–96、98–101,原書 p.214–228)。
> 內容:①以日支(夫妻宮)喜忌算契合指數 ②六神論婚姻 ③不利婚姻的神煞與日柱 ④四柱地支互相比對合婚法 ⑤夫妻婚配要點。

## 一、以夫妻宮(日支)論兩人契合指數

原理:夫妻宮就是「日支」。看日支是命盤的用神、閒神還是忌神,再看它受沖、受合的情形,即可為配偶助力打一個「契合分數」。

| 情形 | 日支狀態 | 論斷 | 契合/助力分數 |
|------|----------|------|----------------|
| 1 | 為用神、不受沖合 | 配偶有助力 | 約 80 分 |
| 2 | 為用神、被閒神沖 | 配偶有助力,但有時不是很甘心幫忙 | 約 60–70 分 |
| 3a | 為用神、被忌神沖 | 夫妻間聚少離多,或晚婚,助力減弱 | 約 60 分 |
| 3b | 用神沖忌神(如子為用神沖午忌神) | 夫妻間助力仍大,可以早婚 | 約 90 分 |
| 4a | 為用神、被合化為忌神 | 助力打折 | 僅約 60 分 |
| 4b | 為用神、被合化仍為用神 | 助力增加 | 約 90 分 |
| 5 | 為閒神 | 相處時好時壞;以現代社會來看還算可以,想更好就要兩人共同努力經營婚姻 | 平 |
| 6 | 為忌神、不受沖合 | 夫妻間較無助力,平凡過一生 | 低 |
| 7 | 為忌神、受閒神沖 | 契合度明顯偏低,沒有太大助力 | 約 10 分 |
| 8a | 為忌神、被用神沖 | 較無助力,但沖掉忌神反而有一點幫助 | 約 20 分 |
| 8b | 忌神沖用神 | 配偶無助力,婚後各自發展事業 | 負 80 分 |
| 9a | 為忌神、被合化為用神 | 相害力減少,仍有助力 | 約 50 分 |
| 9b | 為忌神、被合化仍為忌神 | 相阻力增加 | 負 80 分 |
| 10 | 夫妻宮「得位」且為用神(男日支坐正財、女日支坐正官) | 夫妻間助力最大 | 100 分 |
| 11 | 夫妻宮「得位」但為忌神 | 配偶雖無大助力,但能安守本分、兼顧家庭 | 約 50 分 |
| 12 | 夫妻宮被「爭合」(如巳日受申月、酉時兩邊來合) | 婚姻會三心二意,易有感情糾紛 | 不吉 |

## 二、六神論婚姻(以女命為主)

命局中某類十神過多、關鍵十神無氣,都是婚姻的警訊:

| 條件 | 論斷 |
|------|------|
| (1) 官殺多(兩個以上)且日主無氣 | 命中婚姻緣薄,同時容易受丈夫欺負 |
| (2) 印多(三個以上)且官殺無氣 | 一生為女強人,較不得夫緣,丈夫成就平平;夫妻應多溝通、互相激勵共同打拼 |
| (3) 比劫多且官殺無氣 | 一生為女強人,較不得夫緣;若有財星生官,不在此限 |
| (4) 食傷多且無財生官 | 一生為女強人,較不得夫緣,難有好婚姻——學著稍微放手 |
| (5) 財、殺、官多且日主無氣 | 為夫辛勞,丈夫卻不感激;若日主有氣(有祿刃),不在此限 |
| (6) 神煞:八字中有下列三個以上(男女同論) | 見下節【桃花、華蓋、孤辰、寡宿】 |

## 三、不利婚姻的神煞與日柱組合

命帶下列狀況者,婚姻易有波折;書中強調:這類命「唯有多說好話、多做好事」才能改變婚姻不美的狀況。

| 條件 | 說明 |
|------|------|
| (1) 女命忌庚子日、壬子日、乙卯日、辛酉日;男命忌庚申日 | 難有好婚姻,有可能走向離婚 |
| (2) 女命忌天干三透比劫 | 天生個性不易屈服,好強,婚姻難美 |
| (3) 男女命忌丑戌未三刑入日支 | 雙方都恃勢、太自信、欠考慮而種下敗因 |
| (4) 男女命忌亡神、劫煞沖日支 | 婚姻易有外力(第三者、外界因素)介入,應提防 |
| (5) 男女命忌華蓋坐日支 | 夫妻信念、信仰不同,造成婚姻不美 |
| (6) 男女命忌孤辰或寡宿坐日支 | 個性孤僻、聚少離多之象,婚姻不美 |
| (7) 女命忌「本主同宮」(如庚子年逢庚子日、甲午年逢甲午日) | 因欠缺溝通造成婚姻不美,請加強溝通 |
| (8) 女命忌多合(三合以上) | 被太多人事物絆住,毫無自由,婚姻自然不好 |
| (9) 女命忌月支沖日支 | 婚姻生活缺乏浪漫(沒有瓊瑤小說式的情調),唯有相互體諒才有好結局 |

## 四、四柱地支互相比對合婚法(合婚判斷)

作法:把男女雙方命盤的**四柱地支**並排,「年對年、月對月、日對日、時對時」逐柱比對其合(三合或六合)、沖、害、同(同氣)關係;另外日柱與月柱還要**交叉比對**(男日支↔女月支、男月支↔女日支),看誰遷就誰。

比對架構:

| | 年柱地支 | 月柱地支 | 日主(日支) | 時柱地支 |
|---|---|---|---|---|
| 男命地支 | ↕ 與女年支比對 | ↕ 與女月支比對(並與女日支交叉比對) | ↕ 與女日支比對(並與女月支交叉比對) | ↕ 與女時支比對 |
| 女命地支 | ↕ | ↕ | ↕ | ↕ |

### 各柱比對結果對照表

**年柱(看雙方家長、家庭背景)**

| 關係 | 論斷 |
|------|------|
| 合(三合或六合) | 雙方家長會互相走動、往來 |
| 沖 | 雙方家庭背景不同 |
| 害 | 雙方家長互不往來 |
| 同(同氣) | 雙方家庭背景一致 |
| 無 | 雙方家長互不往來 |

**月柱(看兩人溝通與互動)**

| 關係 | 論斷 |
|------|------|
| 合 | 男女雙方心心相印,好溝通,能達到共識 |
| 沖 | 互動多且刺激,但容易爭吵、意見多 |
| 害 | 相敬如賓,也容易相敬如「冰」,彼此會藏秘密 |
| 同 | 處世態度一致、能協調;但剛認識就覺得超了解彼此,熟了反而沒刺激感 |
| 無 | 沒有好壞之分 |

**日主/日支(看親密關係)**

| 關係 | 論斷 |
|------|------|
| 合 | 情被合住,性生活協調、圓滿 |
| 沖 | 性生活刺激,但也容易要求對方 |
| 害 | 睡覺時間不同、同床機會不多,性生活不美滿,除非小別勝新婚 |
| 同 | 性生活協調、美滿 |

**時柱(看子女教養觀)**

| 關係 | 論斷 |
|------|------|
| 合 | 夫妻對孩子的管教、觀念一致 |
| 沖 | 對孩子的教養觀念不同,會有意見 |
| 害 | 教養觀念意見不一,甚至各用各的方法 |

**男女互動(交叉比對:誰合誰、誰沖誰)**

| 關係 | 論斷 |
|------|------|
| 男合女 | 男方比較會為女方著想,會尊重、會禮讓 |
| 男沖女 | 男對女越看越討厭,意見很多、互不相讓 |
| 男害女 | 男對女越來越不溝通,無言相對 |
| 女合男 | 女方比較會為男方著想,會尊重、會禮讓 |
| 女沖男 | 女對男越看越討厭,意見很多、互不相讓 |
| 女害男 | 女對男越來越不溝通,無言相對 |

## 五、夫妻婚配要點(第五節)

用途:未婚者選擇對象的重要參考;已婚者可印出兩人命盤(最好兩個縮小版並列)一一對照印證,作為日後相處之道,使感情昇華。

1. **夫妻日干成陰陽組合為吉**:夫日干為陽干(甲丙戊庚壬),妻日干配陰干(乙丁己辛癸)為佳;反之亦然。兩人日後能互相扶持,雖有意見但很快就妥協。
2. **夫妻日干成天干五合**:兩人日干恰為甲己、乙庚、丙辛、丁壬、戊癸之一(不論誰是哪一字),彼此有幫助、興趣志同道合,是很好的組合。
3. **夫妻喜忌神不宜相同(互補為佳)**:如夫喜神為木火、忌神為金水,則妻以「忌木火、喜金水」為最佳——雙方喜忌不同可產生互補作用,不會因一方工作事業不順而拖累婚姻。
4. **夫妻喜忌神相同時**:如夫喜木火忌金水,妻也喜木火忌金水——不能產生互補作用,容易因工作事業等不順而一起受影響,波及婚姻。
5. **夫妻日支不宜相同**:如夫日支是「子」,妻日支也是「子」,兩人自主意見都太多,較易發生不和、衝突或生氣。
6. **甲午、庚子、乙巳、辛亥等日生者**:言談舉止風趣多情,易有婚姻不美滿或離婚之象(先看落在哪一方,要標明)。
7. **夫之日支與妻之月支成六合或三合**:老公較會遷就妻子的意見。
8. **妻之日支與夫之月支成六合或三合**:妻子較會聽從丈夫的意見。
9. **夫妻雙方日支互成六合或三合**:彼此都喜歡對方、疼惜對方,意見上也比較容易取得一致。
